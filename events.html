Events not showing?

When you create an event in Google Calendar, choose The Cigar Club in the calendar dropdown—not your personal calendar. Only events on that calendar appear here.

  1. In Google Calendar, confirm The Cigar Club is checked in the left sidebar.
  2. Click the + button → Event, then set the calendar dropdown to The Cigar Club before saving.
  3. Open calendar Settings → Access permissions and enable Make available to public with See all event details.
  4. If your events live on a different calendar, add that calendar ID to embedCalendarIds in js/calendar-config.js.
