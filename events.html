Events not showing? Staff setup guide

Google only publishes events that are saved on The Cigar Club calendar with public visibility. Tasks and private events will not appear on the website.

  1. Click Add event to club calendar above (do not use your personal calendar).
  2. When editing an event, set visibility to Public (not Private).
  3. In calendar settings, enable Make available to public → See all event details.
  4. If events are on another calendar, add its ID to embedCalendarIds in js/calendar-config.js.
